<HEXPATTERNS> => i16{0x8000..0xFFFF}

    (<A> <B> <C> <D>) ||
    (<E> <F>)
)

- nonterminal charset A-Z_-0-9
- one rule per line
- newlines as whitespaces are allowed in brackets
- <A> <B> || <C> <D> || <E> <F> has association <A> (<B> || <C>) (<D> || <E>) <F>
- range X..Y order does not matter so 3..5 == 5..3 and both bounds are inclusive
- no chars, only strings
- include all ascii escape characters in string + \xXX hex escape + \0 + \uXXXX
- entrypoint nonterm is ENTRYPOINT
